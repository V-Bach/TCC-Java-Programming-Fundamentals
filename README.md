# Java Programming Fundamentals (Lập trình Java Căn bản)

Chào mừng bạn đến với kho lưu trữ **Java Programming Fundamentals**. Đây là nơi tổng hợp toàn bộ tài liệu, lý thuyết, và bài tập thực hành dành cho môn học Lập trình Java từ những khái niệm cơ bản nhất cho đến các chủ đề nâng cao.

## 📖 Mục tiêu cốt lõi
Kho tài liệu này được xây dựng nhằm giúp người học:
- Nắm vững cú pháp, từ khóa và các cấu trúc cơ sở của ngôn ngữ Java.
- Hiểu sâu và vận dụng tốt phương pháp Lập trình Hướng Đối Tượng (OOP).
- Làm quen với cấu trúc dữ liệu cơ bản (Collections Framework) và lập trình Generic.
- Mở rộng kiến thức với các tính năng hiện đại và nâng cao như Đa luồng (Multi-threading), Input/Output Stream, và Lambda & Stream API (Java 8+).

---

## 🗂️ Cấu trúc chương trình học

Dự án được chia thành 9 chương, tương ứng với lộ trình học chuẩn xác nhất:

* **[Chương 1: Tổng quan về máy tính và ngôn ngữ Java (Overview) ](Chapter1/Chapter%201.md)**
  - Các khái niệm cơ bản về chương trình máy tính.
  - Lịch sử, đặc điểm của Java và Máy ảo Java (JVM).
  - Cách cài đặt môi trường phát triển (JDK, IDE).

* **[Chương 2: Biến, Hằng và Kiểu dữ liệu (Variables, Constants, Types)](Chapter2/Chapter%202.md)**
  - Phân loại biến, hằng số, quy tắc đặt tên (Naming convention).
  - Các kiểu dữ liệu nguyên thủy và tham chiếu.
  - Các toán tử trong Java (số học, logic, so sánh).

* **[Chương 3: Cấu trúc điều khiển và Mảng (Control Structures and Arrays)](Chapter3/Chapter%203.md)**
  - Cấu trúc rẽ nhánh: `if..else`, `switch..case`.
  - Cấu trúc lặp: `for`, `while`, `do..while`.
  - Làm việc với Mảng (Arrays) và kiểu liệt kê Enum.

* **[Chương 4: Phương thức và Xử lý ngoại lệ (Methods and Exception Handling)](Chapter4/Chapter%204.md)**
  - Khai báo và sử dụng phương thức (Static & Instance).
  - Tham số, nạp chồng phương thức (Method Overloading).
  - Concept về Ngoại lệ (Exceptions) và cách sử dụng cơ chế `try..catch..finally`.
  - Tạo tài liệu Javadoc.

* **[Chương 5: Lập trình Hướng Đối Tượng (Object-Oriented Programming)](Chapter5/Chapter%205.md)**
  - Các tính chất cốt lõi: Đóng gói, Kế thừa, Đa hình, Trừu tượng.
  - Hàm khởi tạo (Constructor), Thuộc tính và Phương thức.
  - Class trừu tượng (`abstract`) và Giao diện (`Interface`).

* **[Chương 6: Lập trình Generic và Collections (Generic and Collections Framework)](Chapter6/Chapter%206.md)**
  - Generic Class và Generic Method.
  - Giới thiệu Java Collections Framework: `List`, `Set`, `Map`.
  - Cách duyệt và thao tác dữ liệu qua `Iterator` và vòng lặp `for-each`.

* **[Chương 7: Luồng dữ liệu (Input/Output Data Stream)](Chapter7/Chapter%207.md)**
  - Khái niệm về Data Flow.
  - Thao tác I/O với file nhị phân (Binary data).
  - Thao tác I/O với file văn bản (Text data).
  - Đọc/Ghi đối tượng (Object Data).

* **[Chương 8: Lập trình Đa luồng (Multi-threading)](Chapter8/Chapter%208.md)**
  - Phương pháp khởi tạo luồng (`Thread` class và `Runnable` interface).
  - Đồng bộ hóa các luồng (Synchronization và Locks).
  - Ứng dụng Framework Fork-Join trong tính toán bất đồng bộ.

* **[Chương 9: Lambda & Stream API (Java 8+)](Chapter9/Chapter%209.md)**
  - Khái niệm Functional Interface (`@FunctionalInterface`).
  - Biểu thức Lambda (Lambda expressions) & Method References.
  - Ứng dụng `Stream API` để xử lý dữ liệu nhanh gọn.
  - Lớp `Optional` để xử lý null một cách an toàn.
